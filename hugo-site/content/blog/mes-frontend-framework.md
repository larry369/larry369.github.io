## 背景

为了降低 MES 系统的外部采购成本，公司决定自製 MES 系统。我负责设计前端 UI 基础框架。

## 框架设计原则

1. **元件化** — 所有 UI 元件独立开发，方便复用
2. **多国语系** — 支援繁中、简中、英文切换
3. **模组化选单** — 根据使用者权限动态显示

## 技术选型

- 框架：Angular
- 状态管理：NgRx
- 国际化：ngx-translate
- UI 元件库：自建

## 成果

框架上线后，新功能页面的开发时间减少了约 40%。
